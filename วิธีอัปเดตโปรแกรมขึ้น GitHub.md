# วิธีอัปเดตโปรแกรมขึ้น GitHub (ไม่ต้องอัปโหลดเองทุกครั้ง)

เนื่องจากแซนด์บ็อกซ์ของ Claude เข้าถึง GitHub โดยตรงไม่ได้ (เครือข่ายถูกจำกัดด้วย allowlist)
วิธีที่ยั่งยืนที่สุดคือใช้ **GitHub Desktop** (โปรแกรมฟรี ไม่ต้องพิมพ์คำสั่งเอง)

## เหตุผลที่แนะนำ GitHub Desktop

หลังติดตั้งครั้งเดียว ทุกครั้งที่อัปเดตแค่กด "Commit" แล้วกด "Push" ไม่กี่วินาทีจบ
ไม่ต้องอัปโหลดไฟล์ทีละไฟล์ผ่านเว็บอีกต่อไป และเพราะ GitHub Desktop เป็นแอปทั่วไป
(ไม่ใช่ terminal ที่ถูกจำกัดสิทธิ์) Claude สามารถควบคุมหน้าจอกดปุ่มพวกนี้ให้แทนได้ในอนาคตด้วย

## ขั้นตอนติดตั้งครั้งแรก

1. ติดตั้ง GitHub Desktop (ฟรี): https://desktop.github.com
2. เข้าสู่ระบบด้วยบัญชี GitHub ของตัวเอง
3. เลือก "Add Local Repository" แล้วชี้ไปที่โฟลเดอร์ `D:\ClaudeCowork\ConsDocx`
4. กด "Publish repository" — **ไม่ต้องติ๊ก** "Keep this code private" เพราะตกลงกันไว้แล้วว่า
   ให้เป็น public ได้ (เพื่อใช้ GitHub Pages แบบฟรี)

## หลังจากนั้น (ทุกครั้งที่มีการอัปเดตโปรแกรม)

1. เปิด GitHub Desktop ขึ้นมา จะเห็นรายการไฟล์ที่เปลี่ยนแปลง
2. กด "Commit to main"
3. กด "Push origin"
4. เว็บไซต์ (GitHub Pages) จะอัปเดตให้อัตโนมัติหลัง push ไม่กี่นาที

## หมายเหตุ

- ถ้าอยากให้ Claude กดปุ่มพวกนี้แทนในอนาคต ต้องอนุญาตให้ Claude เข้าถึงแอป GitHub Desktop
  ผ่านฟีเจอร์ควบคุมหน้าจอ (computer use) ก่อน
- Token แบบ fine-grained ที่เคยสร้างไว้ก่อนหน้านี้ (ใช้ push ผ่าน git command ไม่สำเร็จเพราะ
  เครือข่ายแซนด์บ็อกซ์ถูกบล็อก) แนะนำให้ไป revoke ทิ้งที่ GitHub Settings ได้เลย เพราะไม่ได้ใช้งานจริง
