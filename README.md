# โปรแกรมทำเอกสารโครงการก่อสร้าง (ConsDocx)

เว็บแอปสร้างเอกสารราชการสำหรับโครงการก่อสร้าง กรอกข้อมูลครั้งเดียวแล้วสั่งออก .docx ได้ทุกฉบับ

**เปิดใช้งาน: https://dstaokung.github.io/ConsDocx/** — v1.0

## เอกสารที่รองรับ

| หมายเลข | ชื่อเอกสาร |
|---|---|
| 1 | ขออนุมัติโครงการก่อสร้าง |
| 2 | คำสั่งแต่งตั้งคณะกรรมการ |
| 3 | รายงานการประชุมแบบรูปรายการ |
| 4 | บันทึกรายงานแบบรูปรายการ |
| 5 | นำส่งเอกสารการขออนุมัติ |

## วิธีใช้

1. เปิดโปรแกรมที่ลิงก์ด้านบน
2. กรอกข้อมูลโครงการในหน้าหลัก (ชื่อโครงการ, หน่วยงาน, งบประมาณ ฯลฯ)
3. เลือกเอกสารที่ต้องการ แล้วกด **สร้างเอกสาร** — ได้ไฟล์ .docx ทันที
4. กด **สร้างทุกฉบับ** เพื่อออกเอกสารทั้ง 5 รายการพร้อมกัน

## โครงสร้าง repo

| ตำแหน่ง | หน้าที่ |
|---|---|
| `index.html` | หน้า redirect ไปยัง `forms/main.html` |
| `forms/main.html` | หน้าหลัก — กรอกข้อมูลโครงการและเลือกเอกสาร |
| `forms/<N>.<ชื่อเอกสาร>/` | ฟอร์มและ template-data ของเอกสารแต่ละฉบับ |
| `templates/` | ไฟล์ .docx ต้นแบบสำหรับแต่ละเอกสาร |
| `shared/` | ไลบรารีกลาง (JSZip, docx.lib, ตราครุฑ ฯลฯ) |
